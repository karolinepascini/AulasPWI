TAGS HTML E SUAS FUNÇÕES

TAGS FUNÇÕES
title Define o titulo do elemento
h1,h2,h3... Usada para titúlos,define de acordo com o numero da frente o tamanho que o titúlo será.
p Cria um parágrafo.
br Usado para quebrar linha
b Deixa as letras em Negrito,na versão html 4.
strong Deixa as letras em Negrito,na versão html 5.
i Deixa as letras em Itálico.
hr Cria uma linha.
ol Cria uma lista ordenada
ul Cria uma lista não ordenada
li Tópicos de uma lista, feita na tag ul e ol
header Definem um cabeçalho,todo conteúdo que estiver dentro dela estará no cabeçalho.
article Definem um artigo,usadas para separar o conteúdo da página.
alt Define um texto alternativo e, por isso, é muito utilizado em imagens.


TAGS CSS FUNÇÕES DAS TAGS
id Atribui um id, usado para identificar um único elemento que não pode se repetir.
style Permite colocar um estilo (elemento CSS) na tag,usado para personalizar.Antes de colocarmos cor,fonte,tamanho devemos colocar o style para abrir a chave das tags,ele é um tipo de body.
class Atribui uma classe ao elemento, pode ser usada em um ou mais elementos.
